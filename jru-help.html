Jabber roster utility help

Form Fields

Username - this is the name that you signed up to a Jabber server with. If your full Jabber ID is FunkyMan7@jabber.com, your username is FunkyMan7.

Password - this is simply your password - the combination of characters that gives only you access to your Jabber account.

Server - this is the jabber.com portion of your FunkyMan7@jabber.com Jabber ID.

Port - this should generally be left at 5222, unless you know differently for sure.

Transport Address - the transport or agent is a part of the Jabber server that translates Jabber messages into those to other messaging systems, like AIM or ICQ, and vice versa. The transport address tells the Importer which transport you are registered with so you can subscribe to and contact the users that are being imported. You must be registered with a transport to see the contacts you are importing; each Jabber client does it differently. Generally, enter aim. or icq. as the transport addresses for AIM and ICQ, respectively. For instance, if you are registered on jabber.com, your AIM transport address is probably aim.jabber.com.

Type of buddy list or configuration file - this tells the Importer which program that you are trying to import contacts from. This is necessary because of the different formats different programs store their configurations in.

The empty field and "Browse" button - Click "Browse" to select the file that contains the contacts list or configuration file that goes with the program you are importing from. For more info on which files go with which programs, see below.

IM programs and their files

AOL Instant Messenger - AOL's official client for Windows and for Linux. To obtain these list files, you must sign into your account using the program, and select from the "File" menu to "Export" or "Save" your buddy list. Save these settings and use the file that you obtained from this procedure; they usually end with ".blt" .

Gaim - Similar to AOL's official clients, you must choose "File" and "Export" your buddy list. These files usually end in ".buddy"; use the exported file as the one to upload.

GnomeICU - You do not need to do anything to export files from GnomeICU. The configuration file necessary is located usually in /.gnome/ directory and is called GnomeICU.

mICQ - Similar to GnomeICU, the configuration file necessary is generally located in your home directory and is called .micqrc -- upload this file.

ICQ99b - ICQ 2000b - These files are saved in a subdirectory called something like "Db99" of the directory you installed ICQ in. Inside this directory, the file named .dat is the one that you should upload. (For instance, if your UIN is 988633, the file is 988633.dat) Be patient; these files are generally large in size.

ICQ2001/2002 - These files are saved in a subdirectory called "Contact List" of the directory you installed ICQ in. Inside this directory, choose a recent .clb file to upload.

Licq - Use this small Perl script and copy the resulting list directly to the roster changes text field.
