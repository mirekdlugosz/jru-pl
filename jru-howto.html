Jabber roster utility HowTo

How to import ICQ/AIM contact lists
  1. Launch you favourite Jabber client
  2. Register with the transport of choice - remember its name (for example aim.some.jabber.server)
    Important: This has to be done before importing your contacts in step 5
  3. Fill out the roster utility's "Convert buddy list to roster changes" form fields, click "Convert file"
  4. Once the file has been converted check the resulting "Roster changes" and modify if necessary
  5. Fill out the roster utility's "Server information" form field, click "Send roster changes to Jabber server"

How to migrate from one Jabber server to another
  1. Launch you favourite Jabber client
  2. Register a new account on the destination Jabber server
  3. Fill out the roster utility's "Server information" form field (with your old account's authorization information), click "Get roster from Jabber server"
  4. Check out the resulting roster changes
  5. Fill out the roster utility's "Server information" form field (with your new account's authorization information), click "Send roster changes to Jabber server"

How to migrate contacts from one transport to another
Perhaps you find the transport(s) you are currently using too unstable or unreliable. Perhaps you want to use the new ICQv7-t for your ICQ contacts instead of AIM-t. Just migrate your contacts: With Jabber you have full flexibility, you can even use transports set up on servers you do not habe a Jabber account on.
  1. Launch you favourite Jabber client
  2. Unregister with the current transport, register with the new transport - remember its name (for example aim.some.jabber.server)
    Important: This has to be done before importing your contacts in step 9
  3. Fill out the roster utility's "Server information" form field, click "Get roster from Jabber server"
  4. Copy the resulting roster changes to your favourite text editor, delete all contacts that do not have to be changed
  5. Duplicate the remaining contacts and replace all leading "+" by "-" on the original contacts
  6. Replace the old transport's address with the new transport's address on the remaining contacts (the ones starting with "+")
  7. Now you have one block of contacts with a leading "-" (this means all contacts using the old transports are going to be deleted from your roster) and a block with a leading "+" (the contacts using the new transport)
  8. Copy the text back to the roster changes form field
  9. Fill out the roster utility's "Server information" form field, click "Send roster changes to Jabber server"
